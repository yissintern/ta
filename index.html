🔒 개인정보 수집·이용 동의

1. 수집 목적

YISS/WAY 프로그램 조교 출퇴근 관리 및 근로장학금 산정

2. 수집 항목

  • 학번, 이름 (본인 식별)
  • GPS 위치 좌표 (출근·퇴근 시 각 1회)
  • 출퇴근 시간

3. 보유 및 이용 기간

해당 프로그램 종료 후 6개월까지 보유하며, 장학금 지급 완료 후 GPS 좌표 데이터는 파기합니다.

4. 동의 거부 시 불이익

동의하지 않으실 경우 본 시스템을 통한 출퇴근 체크인이 불가하며, 별도의 수기 출석 절차로 대체됩니다.

5. 안내 사항

  • GPS 위치는 출근·퇴근 버튼 클릭 시에만 수집되며, 실시간 추적은 하지 않습니다.
  • 수집된 위치 정보는 캠퍼스 내 출근 여부 확인 목적으로만 사용됩니다.
  • 수집된 개인정보는 제3자에게 제공되지 않습니다.

📍 브라우저 위치 권한 허용 필요

동의 후 브라우저에서 "위치 접근 허용" 팝업이 뜨면 반드시 허용을 눌러주세요. 차단 시 출퇴근 체크인이 불가합니다.
